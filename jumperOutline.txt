Object: director 
Responsibility: calls all the others class
Behavior: It uses the methods stored from the other classes to execute the game  and take away parts of parachute based on wrong guesses.

Object: Player
Responsibility: store all the game information tied to the player including the parachute
Behavior: keep track of guesses and remove section of the parachute

Object: terminal service
Responsibility: oversee displaying what stage the jumper 
Behavior: Displays how much loss of parachute the player has as the game goes on. 

Object: Word
Responsibility: Be responsible for generating and making the word. Also track the word is important

States:
    - words list   
    - correct for the user to guess 
    - current word for the user to see where they are at in the game 
Behavior: 
    - Pulls a list from a txt file stores it into an array.
    - Selects a word from the list for the user to guess.
    - Fills in the correct guesses of the word in the letter spaces
